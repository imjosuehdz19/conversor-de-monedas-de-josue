

# Conversor de Monedas en Java por el Challegue Alura Latam

## Descripción
Este proyecto permite evaluar 6 tipos de cambio utilizando la API **ExchangeRate-API**, para ello se necesitó crear en primera instancia una API Key (requerirás crear tu propia API Key).
Puedes clonar el repositorio y editar unicamente la API Key que hayas creado en tu IDEA de preferencia (se uso IntelliJ en este caso) .
